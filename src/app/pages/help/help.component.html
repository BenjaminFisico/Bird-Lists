Este proyecto fue creado desde cero utilizando el framework javacript Angular v.16.2.11 y aprovechando el cdk de Angular para crear el sistema que permite arrastrar y soltar listas o tareas.

En caso de que tengas alguna crítica o mejora para este proyecto es muy bienvenida, por favor contáctame.
¿Qué es Bird Lists?

Es una aplicación web que permite gestionar nuestras tareas. Es decir, es un proyecto de TO DO LIST más complejo ya que permite diferentes niveles de personalización e incluso la posibilidad de guardar un proyecto para seguir trabajando en él más tarde.

Este tipo de aplicaciones suelen ser muy útiles para organizarnos mejor y así lograr una mayor productividad en consecuencia.

Aprovecha el potencial de Bird Lists para enfocarte en una sola tarea a la vez, olvídate de eso de memorizar cada tarea todo el tiempo.

¿ Por qué?

Bird Lists nació de la idea de simplificar el proceso de crear listas de tareas en Trello permitiendo almacenar los diferentes proyectos en la computadora del usuario.

Y para adicionalmente agregar algunos cambios que me parecen interesantes para el uso de las aplicaciones TO DO.

¿Cómo se usa?

Al abrir la aplicación te encontrarás con dos botones.

Uno que permite crear un nuevo proyecto eligiendo un título y los diferentes colores por defecto para las listas y tareas.

El otro sirve para cargar un proyecto en formato JSON que hayamos guardado con anterioridad.

Se eligió el formato JSON porque este es muy fácil de editar desde cualquier editor de texto.

Una vez que ya tengas un proyecto iniciado puedes añadir tantas listas de tareas como gustes apretando el botón de + Añadir otra lista.

Cuando ya tengas las listas de tareas que querías puedes agregar tantas tareas como gustes apretando el botón que aparece dentro de cada lista y que dice Añadir una tarjeta.

En la esquina superior derecha del proyecto nos encontraremos con el botón guardar que como su nombre indica nos permite guardar el proyecto en el almacenamiento personal del usuario.

Características adicionales

En cada lista de tareas tenemos diferentes opciones a las cuales podemos acceder haciendo click sobre el botón ... que se encuentra en la esquina superior derecha de cada una de estas.

Dichas opciones incluyen: Renombrar la lista, duplicar la lista (es decir crear una lista idéntica con sus propiedades y tareas internas), cambiar sus propiedades es decir colores y/o eliminar definitivamente la lista.

En la esquina superior izquierda de cada lista de tareas hay un icono, desde el el cual, podemos arrastrar la lista manteniendo presionado el click y así intercambiar su posición con la de otra lista de tareas.

Al hacer click sobre una tarea individual podemos cambiar sus colores, eliminar la tarea o habilitar/deshabilitar el cuadro para indicar que la tarea fue completada.

También podemos arrastrar las tareas para intercambiar su orden con otras.

Próximos cambios planeados

En el futuro voy a agregar las siguientes características:

  1. Al agregar una nueva tarea, un botón que nos permita ingresar emojis y así mismo una forma rápida de agregar emojis con texto.
  2. Que al indicar que se completó una tarea esta se envíe automáticamente al fondo de la lista y que exista la posibilidad de desactivar esta opción en cada lista de tareas.
  3. Que se pueda agregar una barra de progreso para cada lista que nosotros queramos y dicha barra de progreso se complete según las tareas completadas en esa lista de tareas específica.
  4. Una barra de proceso para cada proyecto creado que nos ayude a dimensionar la cantidad de tareas del total ya han sido completadas.
  5. Que sea posible interactuar por toda la página utilizando solo el teclado para mejorar mucho la UX.
  6. Una nueva forma de organización automática de tareas que consista en la fecha en la que las mismas deben ser realizadas o en las que las mismas hayan sido creadas.
